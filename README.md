Rooset
========

Rooset is a democratic structured decision system.


Dependencies
=============

A Linux environment with the following:
  - Postgresql  
  - [EventStore](https://geteventstore.com) 3.9 +
  - Clang 3.8 +
  - Java 8 +
  - NodeJS 6.9 +
  - CMake 3.0 +
  - bash


Install
=========

```
  $ git clone https://github.com/ddouglascarr/rooset
  $ cd rooset
  $ git submodule update -- init
  $ ./configure
```

Tests
=========

```
  $ ./run-all-tests
```
